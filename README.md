# PMDD_RNA-seq_analyses

A project completed as part of the requirements for BINF*6210 at the University of Guelph.
This project analyses mouse RNA-seq data generated by Marrocco et al., 2018. They examined differential expression of genes between wild-type mice and mice heterozygouse for Val66Met in BDNF, in response to treatment with estradiol. This model mimics the abnormal reaction of women with PMDD to fluctuating levels of hormones during the menstrual cycle.
This project uses code adapted from Law et al. (2018) and Law et al. (2020).
